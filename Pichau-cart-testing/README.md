# Projeto de teste - Feature de Carrinho de Compras (E-commerce Pichau)
### Objetivo: Testar funcionalidade de carrinho de compras de um e-commerce real.

# Atividades realizadas:
- Criação de cenários de teste;
- Escrita de casos de teste;
- Execução de testes manuais;
- Registro de evidências;
- Identificação de bug;
- Criação de bug report no Jira;
- Vinculação do bug à user story.

## Ambiente:
Produção (site público da Pichau)

## Ferramentas utilizadas:
- Trello;
- Google Sheets;
- Jira.

## Resultados:
- 6 cenários de teste;
- 11 casos de teste;
- 1 bug (simulado) identificado na atualização do carrinho após remoção de item.
